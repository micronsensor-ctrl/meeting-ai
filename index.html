⚡ วิธีเปิดใช้ "ปุ่มเดียวจบ"
ทำครั้งเดียว 5 นาที:

1. เปิด platform.claude.com → ล็อกอิน Gmail เดิม
2. เลือก Individual → Buy credits → USD 5 (~180 บาท ใช้ได้ ~150 ประชุม)
3. ไปเมนู API Keys → Create Key → copy (ขึ้นต้น sk-ant-)
4. กลับแอปนี้ → กด "⚙️ ตั้งค่า AI" มุมขวาบน → วาง Key → บันทึก

เสร็จแล้วขั้นที่ 3 จะกดปุ่มเดียว AI สรุปในแอปเลย ไม่ต้องสลับหน้าจอ

ยังไม่พร้อมจ่าย? ใช้ "วิธีฟรี" ที่ซ่อนอยู่ในขั้นที่ 3 ได้เหมือนเดิม
💾 ข้อมูลหายไหมถ้าปิดเว็บ?
ไม่หาย — ระบบบันทึกอัตโนมัติในเครื่อง ปิดแล้วเปิดใหม่ข้อมูลยังอยู่
ถ้าอยากเริ่มใหม่ กดปุ่ม "เริ่มประชุมใหม่" ในขั้นสุดท้าย
📋 Copy/วาง ทำยังไงบนมือถือ?
Copy: กดค้างที่ข้อความ → ลากเลือก → กด "คัดลอก"
วาง: แตะในช่อง → กดค้าง → กด "วาง"
หรือใช้ปุ่ม "📋 วางจากคลิปบอร์ด" ในแอป
🔗 บันทึกแอปไว้หน้าจอหลัก
เปิดเมนู Chrome (จุด 3 จุดมุมขวาบน) → "เพิ่มลงในหน้าจอหลัก"
จะได้ไอคอนเหมือนแอปจริง เปิดได้เร็ว ไม่ต้องพิมพ์ URL

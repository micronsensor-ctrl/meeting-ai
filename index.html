📖วิธีใช้ MeetingAI v5 Free
1. ตั้งค่าข้อมูลประชุม
ชื่อ, วันที่, เวลา, สถานที่, ผู้เข้าร่วม

2. อัดเสียง Notta (75 นาที)
เปิด Notta → Start Recording → ประชุม

3. ถอดข้อความ
Notta → Export → Copy transcript

4. Claude.ai Summarize
• เปิด https://claude.ai
• Copy prompt ตรงนี้ (ปุ่ม Copy Prompt)
• Paste transcript
• Claude จะ summarize → Copy JSON ผลลัพธ์

5. วาง JSON ที่นี่
paste ตรง "วาง JSON Result จาก Claude.ai"

6. รับรอง
ตรวจสอบ → แก้ไขได้ → กด "รับรองรายงาน"

7. ส่ง
Email + Calendar + LINE auto

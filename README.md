# MeetingAI v4 — Trident 1995

Automated meeting notes generator with real-time transcription and AI summarization.

## Quick Start

1. Go to: [GitHub Pages URL will be auto-generated]
2. Open in Android Chrome
3. Insert Anthropic API Key
4. Start recording meeting with Notta
5. Paste transcript → Claude summarizes automatically

## Features

- ✅ 4 input methods (Notta, Google Recorder, upload, manual paste)
- ✅ Real-time Thai/English/Chinese language support
- ✅ Automatic Action Items with assignees and deadlines
- ✅ Google Calendar integration
- ✅ Email distribution
- ✅ Gboard shortcuts for fast note-taking
- ✅ 2-device sync via Google Drive

## Setup

```bash
git clone https://github.com/YOUR-USERNAME/meeting-ai.git
cd meeting-ai
# Open index.html in browser or deploy to GitHub Pages
```

## Deploy

Just push to `main` branch → GitHub Actions auto-deploys to GitHub Pages

## Architecture

- HTML/CSS/JS only (no build step)
- Claude API for summarization
- Whisper API for transcription (optional)
- Works offline with Google Docs
